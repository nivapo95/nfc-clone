### What you will need


* NFC Card reader (I used [ACR122U](https://www.amazon.com/ACS-ACR122U-Contactless-Smart-Reader/dp/B01KEGQFYY))
* A NFC card with existing data to replicate 
* Blank NFC cards (read+write)
